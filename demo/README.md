# 正方形領域の合成視線データ — 可視化デモ

既存研究の数理モデルで **1000×1000 px の正方形領域上の視線（gaze / line-of-sight）データ**を合成し、
ブラウザで可視化する自己完結デモです。

## 数理モデル: Constrained Lévy Exploration (CLE)

> Boccignone, G. & Ferraro, M. (2004).
> *"Modelling gaze shift as a constrained random walk"*, **Physica A** 331, 207–218.

視線移動（サッケード）を「**サリエンシー地形に制約された Lévy フライト**」として生成します。

1. **Lévy フライト** — サッケード振幅は重い裾を持つ分布（`P(l) ~ l^{-(1+α)}`）から引く。
   短い跳躍が大半だが稀に大きく飛ぶ、というヒトの探索的眼球運動の性質を再現。
2. **サリエンシー制約（Metropolis 則）** — 提案された移動先は、より顕著なら必ず受理、
   そうでなければ `exp(Δs/T)` の確率で受理。結果として視線が顕著領域へ自然に吸着する。
3. **固視微動（Ornstein–Uhlenbeck）** — 注視中の微小な揺れを平均回帰過程で表現。
4. **注視時間** — ガンマ分布（平均 ~250 ms）からサンプリング。
5. **サッケード補間** — 注視点間を最小ジャーク速度プロファイルで滑らかに接続し、250 Hz の raw gaze に展開。

すべて乱数シード固定（`seed = 42`）で再現可能。**標準ライブラリのみ**で動作（numpy 不要）。

## 使い方

```bash
# データ生成 + 自己完結 HTML のビルド
python3 demo/generate_los_data.py

# 可視化を開く（ブラウザで開くだけ）
open demo/index.html        # macOS / お使いの環境のブラウザで demo/index.html を開く
```

`generate_los_data.py` は次を出力します。

| ファイル | 内容 |
|---|---|
| `los_data.json` | 注視点列・raw gaze 時系列・サリエンシー格子・統計を含むデータ |
| `index.html` | データを埋め込んだ単一ファイルの可視化（`index.template.html` から生成） |

## 可視化でできること

- **サリエンシー** 背景、**スキャンパス**（サッケード軌跡）、**注視点**（円の大小=注視時間、色=時間順）の重ね表示切替
- 注視密度の **ヒートマップ**
- raw gaze（250 Hz）の **再生**（速度・シーク調整可、サッケード相は橙・固視相は緑で表示）

色・点の意味はページ右下の凡例を参照。
